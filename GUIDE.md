# 📖 Guia de Uso - CodeMaster Kids

## Para Educadores e Pais

### Como usar a plataforma com crianças

#### 1. **Criar uma Conta**
- Acesse a plataforma
- Clique em "Registrar"
- Preencha: nome de usuário, nome completo e idade
- Pronto! Já pode começar a aprender

#### 2. **Entender os Níveis**
A plataforma tem 5 níveis progressivos:

**🟢 Nível 1: Fundamentos Python**
- Variáveis e tipos de dados
- Operações matemáticas
- Estruturas if/else
- Loops com for
- Funções básicas
- **Tempo**: 1-2 semanas

**🟡 Nível 2: Python Intermediário**
- Listas e dicionários
- Loops com while
- Manipulação de strings
- Tratamento de erros
- **Tempo**: 2-3 semanas

**🟠 Nível 3: Algoritmos & Estruturas de Dados**
- Busca e ordenação
- Recursão
- Complexidade de algoritmos
- **Tempo**: 2-3 semanas

**🔴 Nível 4: SQL**
- Criação de tabelas
- SELECT, INSERT, UPDATE, DELETE
- JOINs e consultas complexas
- **Tempo**: 2-3 semanas

**🟣 Nível 5: IA e Machine Learning**
- Introdução a IA
- Datasets
- Modelos simples
- **Tempo**: 2-3 semanas

#### 3. **Sistema de Gamificação**

**⭐ Pontos**
- Cada desafio completado = pontos
- Acumule pontos para desbloquear níveis
- Leaderboard mostra progresso

**🏆 Conquistas (Badges)**
- Primeiro Código
- Mestre em Python
- Assistente SQL
- Explorador de IA
- E muitas mais!

**🔥 Streak**
- Número de dias seguidos aprendendo
- Maiores streaks = mais recompensas
- Mantenha a motivação!

#### 4. **Como Estudar Cada Aula**

1. **Leia o Conteúdo**: Entenda os conceitos
2. **Veja os Exemplos**: Execute o código junto
3. **Pedir Dica**: Se não entender, peça uma dica
4. **Fazer o Desafio**: Aplique o que aprendeu
5. **Conversar com IA**: Tire dúvidas com Claude

#### 5. **Executar Código**

- Cole seu código no editor
- Clique em "Executar Código"
- Veja a saída e erros
- Corrija e tente novamente

#### 6. **Usar o Assistente de IA**

O assistente Claude pode ajudar com:
- Explicar conceitos
- Dar dicas sem revelar a resposta
- Debugar código com erros
- Sugerir melhorias

**Dica**: Faça perguntas claras! "Como faço um loop?" é melhor que "Não entendi"

#### 7. **Integração com Robótica**

Cada aula tem uma seção "Robótica" mostrando:
- Como o conceito se aplica em robôs reais
- Exemplos práticos com Arduino/Raspberry Pi
- Projetos integradores

Exemplo:
```python
# Controlar LEDs de um robô
ligar_led(LED_VERMELHO)
desligar_led(LED_VERMELHO)
```

## 🆘 Troubleshooting

### "Não consigo executar o código"
- Verifique a sintaxe (parênteses, dois-pontos)
- Leia a mensagem de erro
- Peça dica ao assistente

### "O código executa mas não faz o que quero"
- Revise a lógica
- Adicione prints para debugar
- Use o assistente para revisar

### "Estou preso em um desafio"
- Peça uma dica (não spoilers!)
- Releia a aula
- Converse com Claude
- Tente abordagem diferente

### "Qual é o próximo passo?"
- O assistente sugere próximos passos
- Siga os níveis na ordem
- Faça projetos práticos
